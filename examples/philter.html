Philter

Philter is a class which provide several pixel manipulation filters such as blur, sharpen and many more.

This class work on canvases.

The general use is : 

var p = new O876.Philter();
p.run(canvas, filter-name, [options]);
  • filter-name : a string specifying a filter.
  • options : an optional plain object containing all the required options. see examples.

Each filter may requires paremeters. Each example below, shows how to use every filters and what parameters are needed.

Blur

The classical blur effect will blur the image.

  • radius integer, default value = 1 - the blur radius in pixel. The higher value, the more time it takes to blur the image. For radius greater than 1px, a gaussian blur is computed

Sharpen

Another classical effect. Sharpen will accentuate edges. It may sometimes enhance details on a blurry image.

  • more boolean, default value = false - When set to true, the Sharpen effect is even stronger.

Emboss

This filter will give some relief with bumps and hollows.

  • more boolean, default value = false - When set to true, the Emboss effect is even stronger.
  • sobel boolean, default value = false - When set to true, the sobel variant of the emboss filter is used.

Edges

This filter will detect edges and enlight them while darkening everything else.

  • there are no params

Grayscale and Sepia

These filters will "monochromize" the image. The grayscale filter will remove any color, leaving only gray pixels. The sepia filter will colorize the image in a sepia tone.

  • there are no params

Contrast

This filter will change the contrast level. The brighter pixels will become even brighter and the darker, even darker. The effect is reverse when a negative amount of contrast is specified.

  • level integer, default value = 50 - Specifies the amount of contrast. If a positive value is specified, the constrat will increase. If a negative value is specified, the contrast level will decrease leaving a grayish image

Noise

This filter will add noise to the image. That basically means that any pixel will be add to a random value affecting its color.

  • level integer, default value = 50 - Specifies the amount of noise. Please only use positive values.

Negate

This filter will negate all color values.

  • there are no params

Hue, saturation, lightness

This filter will change the hue, saturation, and brightness of all pixels.

  • hue float, default value = 0.0 - This parameter (between -1 and 1) will be added to the pixel hue, to get a new pixel hue value. Visually, this will change the pixel color.
  • saturation float, default value = 0.0 - This parameter (between -1 and 1) will changed the pixel saturation. When reaching higher saturation levels, the pixel color will look vivid, whereas a low saturation value means grayish colors.
  • lightness float, default value = 0.0 - This parameter (between -1 and 1) will changed the pixel lightness. The lower the darker. The hight the brighter...

Kuwahara

Kuwahara is an artistic filter. It modifies a picture, so it will look like a painting. Warning : this filter is rather slow...

  • radius integer, default value = 1 - The size of the brush. The higher value, the larger brush stokes and lower details.
